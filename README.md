$ Bakery Shop project [베이커리 쇼핑몰]

## 🧐 배경

- 스프링 부트와 Jpa, Security 등의 기술스택들을 이용해 개인 포트폴리오를 완성했지만, 관련 지식을 더 숙지할 필요성을 느껴서
  학습 목적으로 시작하게 됐습니다.

## 프로젝트 개발 환경
1.  운영체제 : Windows 10
2.  통합개발환경(IDE) : intellij
3.  JDK 버전 : JDK 11
4.  스프링 부트 버전 : 2.5.2
6.  빌드 툴 : Maven
7.  관리 툴 : GitHub

#### 주요 프레임워크 / 라이브러리
- Java 11
- SpringBoot 2.7.1
- SpringBoot Security
- Spring Data JPA
- QueryDsl

#### DataBase 
- MySql

#### 프론트엔드
- Html/Css
- Thymeleaf
- Bootstrap5

#### API
- OAuth2.0 API, Java Mail
